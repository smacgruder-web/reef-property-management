# Integration Guide: Bringing Stitch Designs into Your App To integrate the "Aquatic Precision" screens into your live application, follow these steps: ## 1. Access the Source Code Each screen you see on the canvas is built with clean, production-ready HTML and Tailwind CSS. - **Action:** Select a screen and click the ** View Code** button in the top toolbar. - **What you get:** You can copy the full HTML structure and CSS classes directly into your frontend project (React, Vue, or plain HTML). ## 2. VS Code: Copy & Paste Instructions Follow these steps to quickly move your designs into your local environment: 1. Open your project in VS Code. 2. Create a new HTML file (e.g., 'dashboard.html'). 3. In Stitch, select the screen you want, click the ' View Code' button, and copy the entire code block. 4. Paste that code into your new file in VS Code. 5. Ensure your project has the Tailwind CSS CDN or a build step configured. 6. Add the Google Fonts links for 'Manrope' and 'Inter' to your section to match the design perfectly. ## 3. Export to Figma If you need to perform further UI refinements or hand off to a design team: - **Action:** Click the **Export to Figma** button in the toolbar. - **What you get:** A high-fidelity Figma file with all layers and design tokens preserved. ## 4. Implementation Workflow 1. **Setup Tailwind:** Ensure your project has Tailwind CSS installed, as all designs rely on its utility-first framework. 2. **Font Selection:** Download and include the 'Manrope' and 'Inter' font families from Google Fonts. 3. **Asset Management:** Any images used in these designs can be downloaded or replaced with your own brand assets. 4. **Dynamic Data:** Replace the placeholder text in the HTML (e.g., "$142.8M") with dynamic data variables from your backend API. ## 5. Documentation Reference Refer to the **Developer Technical Sheet (DOCUMENT_26)** already on your canvas for design tokens, primary color hex codes (#14B8A6), and specific component styling notes.
